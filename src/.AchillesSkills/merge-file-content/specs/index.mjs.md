Skillul merge-file-content aplică modificări la conținutul unui fișier existent utilizând analiză LLM pe baza instrucțiunilor specificate. Skillul returnează conținutul actualizat ca șir de caractere. Skillul depinde de agentul LLM pentru aplicarea modificărilor și de modulul node:fs pentru citirea fișierelor dacă este necesar. Inputul așteaptă un prompt care specifică calea fișierului, conținutul curent și instrucțiunile în formatul "path: /cale, currentContent: text, instructions: text". Skillul returnează conținutul modificat dacă operația reușește, altfel aruncă erori pentru parametri invalizi sau eșecuri ale LLM-ului.
