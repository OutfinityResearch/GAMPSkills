# merge-file-content

## Description
Applies changes to an existing file's content via LLM analysis, such as inserting, replacing, or modifying sections based on instructions. Returns the updated content string.

## Instructions
Take the file path, current content, and change instructions (e.g., resolution text). Use LLM to analyze and apply the modifications to the content. Validate the result to ensure it's valid, and return the revised string.
