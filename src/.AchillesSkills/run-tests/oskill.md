# run-tests

## Description
The run-tests skill executes the project's test suite and reports the outcomes. It runs the configured test command, captures results including pass/fail counts and details on failures, and provides a summary for further action or review.

## Instructions
The LLM initiates the testing process by executing the project's test command in the appropriate environment. It monitors the execution for completion, gathering detailed information about test performance and any issues encountered. The results are analyzed and presented in a clear format, highlighting successes and pinpointing areas that require attention to maintain code quality and functionality.
