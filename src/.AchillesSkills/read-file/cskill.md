# read-file
Reads the content of a file at the specified path.

## Summary
Reads the content of a file at the specified path.

## Input Format
- **filePath** (string): Absolute path to the file.

## Output Format
- **Type**: `string`
- **Success Example**: "File contents..."
- **Error Example**: "Error: File not found."

## Constraints
- Read using UTF-8.
- Return the full file content.
